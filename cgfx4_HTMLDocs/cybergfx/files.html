CyberGraphX V3 41.19 - Release 70a WWW@VGR@Phase5
WWW@VGR
FTP@VGR
FTP@Phase5 		The CGX AGA PPC Driver is now in the archive. This is NOT a AGA(AA) emulator! It needs AGA and a PPC card to work. It will create CGXAGA screen modes using the AGA chipset like a gfx card allowing you to run CGX programs that will use the AGA chipset for display. CGX ENV/settings/file Info Page for info on the new ToolTypes!

CyberVision64/3D Install Disks
THIS IS FOR THE CyberVision64/3D ONLY!
CyberVision64/3D Install disk/file 41.1 (b40) One thing that I don't like about a multi-disk install is that there is never an option to place everything on a HD Floppy (1.76MB) or in one directory on a HardDrive or in RAM:. I have taken both 41.1 install disks for the CV3D and moved them into 1 file and changed and updated the installer to support a 1 HD-Floppy (or HD/RAM:) install. Just uncompress this to where ever and install... no disk swaping. Phase5 might not know about this 1 file archive so if you have any problems with this please email me! vgr@best.com [Once 41.1 (b40) is installed you can use the updates from below to bring V3 up to the current release.]
