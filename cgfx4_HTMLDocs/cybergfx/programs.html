Cool program of the 'Month'
File Name About Where to get
Quake (February-25-98) Quake has been ported to the Amiga by ClickBoom. This is a commercial Game. Here is the Quake Page. Quake File Page
Descent (February-24-98) Descent has been ported to the Amiga from the PC. It seem to work just fine on a CGX screen. ADescent is the file. Here is a link to an "Amiga descent" page by a 3rd party: Amiga Descent page. Also a link to the ADescent Authors page. FTP WWW
Doom (January-16-98) Doom has been ported to the Amiga. There are a number of different ports by many people currently. There is a nice webpage set up that is tracking all the Doom ports. It seem to work just fine on a CGX screen! Doom Page
Myst (December-24-97) Myst has been ported to the Amiga by ClickBoom. This is a commercial Game. If you have Myst there is also a CGX Myst Patch you can get. Here is the Myst Page. Myst File Page
Birdie (September-02-97) Another Very Cool! Tronan hack! Birdie is 'a dirty hack that gives patterns in window borders for cgx 3 users'. 'It's really really cool, unless it crashes you. It's a real life software lottery.' By: Trond Werner Hansen, the guy that has not tried chocolate yet! :) Birdie Web Page
Cyber56khz (April-15-97) Cyber56khz - Allows 56khz samples to be played on CGX V3. This quick `hack' allows CGX V3 owners to play samples ( or use MPEGA ) at 56khz. It sets ENV:CyberGraphX/NOPASSTHROUGH briefly , opens a MULTISCAN:PRODUCTIVTY display , then closes it leaving the AGA DMA intact. Then restores the ENV: variable to its orginal state. By: Dale Kirkwood - kirkwood@enterprise.net WWW@VGR
FTP@VGR
WolfPac (March-26-97) WolfPac is a 3D Pacman clone, so all you have to do is to eat all pills you find in the labyrinth, without being caught by the ghosts. If you have eaten all pills (the pillcounter on the informationpanel in the game shows how much of them are left) you will advance to the next level. By: Guenter Erhart - erhart@onlineloop.com WWW
CyberQT (Feb-27-97) CyberQT is a fast QuickTime animation player for 020+ machines with AmigaOS 3.0 or higher and a graphic board with CyberGraphX or AGA. Plays: Radius CinePak (CVID), Microsoft Video 1 (MSVC), Apple Animation (RLE), Apple Uncompressed (RAW), Apple Video (RPZA), Apple Graphics (SMC), Component Video (YUV2), Intel Raw (YUV9). By: Thore Boeckelmann, tboeckel@uni-paderborn.de FTP WWW
Schnaps (Feb-10-97) 'This simple patch will let you snap text with PowerSnap or SnapV2 (both on Aminet) on hi/truecolor CyberGFX screens. The text you want to snap must be either black or white. I have only tested it with my own CyberVision card, but it should work with all kind of pixelformats, from RGB15 to RGBA32.' By: Trond Werner Hansen, tronhan@stud.ntnu.no, IRC: Tronan
Check out the WorkbenchNG site at http://www.stud.ntnu.no/~tronhan/wbng.html 		WWW
